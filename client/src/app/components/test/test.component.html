Test paneel

Welkom op het tijdige testpaneel en allereerst bedankt dat je mijn website wilt testen! Via de onderstaande knoppen zal het testen een stuk vergemakkelijkt worden.

Nadat je jezelf moderator hebt gemaakt moet je even in en uitloggen. Vanwege de veiligheid wordt rolleninformatie namelijk vanuit de backend meegestuurd tijdens inloggen.

Wanneer je op een knop hebt gedrukt is het aangeraden om even een refresh (f5) te doen zodat de cache van de frontend code geleegd wordt.
Caching wordt voornamelijk gebruikt om het huidige actieve thema te onthouden. Aangezien deze informatie een week lang hetzelfde blijft zou het zonde zijn daarvoor telkens requests naar de backend te sturen.
